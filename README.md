* [Keras 2.3](https://github.com/keras-team/keras) 
* [Matplotlib 2.2](https://matplotlib.org/)
* [Pandas 0.23.4](https://pandas.pydata.org/)
* [Numpy 1.15.4](https://www.numpy.org/)
* [Scikit-learn](https://scikit-learn.org/stable/)
